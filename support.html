Quick answers

For small things that should not require a whole ticket.

My agent is not responding. What should I try first?
Refresh the dashboard, check whether your channel is connected, then ask Clippy Ron to help decide if this needs a ticket.
Where do I set up my agent?
Start in Launchpad. It walks you through name, agent name, first mission, tools, and guardrails.
When should I make a ticket?
If it involves billing, account access, broken container behavior, lost setup, or something private, make a ticket so Robby and Cassie can see the details.
Should I ask Discord or support?
Use Discord for general questions and community ideas. Use tickets for private account-specific problems.
